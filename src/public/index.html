Setup Guide

Client Name

Any label that helps you identify the client internally. Can be the company name or a shorthand like "acme-ops".

SmartLead API Key

  1. Log in to SmartLead
  2. Click your profile avatar (top right) → Settings
  3. Go to the API Keys section
  4. Click Add New API Key, name it, and copy the key
Leave blank if this client doesn't run email campaigns.

HeyReach API Key

  1. Log in to HeyReach
  2. Go to Settings → Integrations → API Keys
  3. Click Generate New Key and copy it
Leave blank if this client doesn't run LinkedIn campaigns.

Slack Bot Token

  1. Go to api.slack.com/apps and open your SalesGlider Slack app
  2. Click OAuth & Permissions in the left sidebar
  3. Scroll down to Bot Token Scopes and confirm these are added:
    • chat:write
    • chat:write.public
    • channels:history
    • im:write
  4. Scroll back up and copy the Bot User OAuth Token (starts with xoxb-)
Same token for all clients if they share the same Slack workspace. Different workspaces need different apps & tokens.

Slack Channel ID

  1. In Slack, create a dedicated channel for this client (e.g. #acme-replies)
  2. Invite the bot to the channel: type /invite @YourBotName in the channel
  3. Right-click the channel name in the sidebar → View channel details
  4. Scroll to the bottom — the Channel ID is listed there (starts with C)
Alternative: right-click channel → Copy link. The ID is the last segment of the URL.

Cal.com Event Type ID

An "event type" is a bookable meeting page on Cal.com — like a Calendly link.

  1. Log in to cal.com
  2. Go to Event Types in the sidebar
  3. If the client doesn't have one yet, create it (e.g. "30-Minute Discovery Call"). Have them connect their Google/Outlook calendar under Apps first — Cal.com will auto-block their real calendar to prevent double bookings.
  4. Click the event type to edit it
  5. Look at the browser URL: cal.com/event-types/123456 — the number at the end is the Event Type ID
  6. Paste that number into the form
Cal.com sends the confirmation email and calendar invite automatically when we create a booking — no extra setup needed.

Voice Prompt

2–4 sentences describing how Gemini should write on this client's behalf. Think tone, style, phrases to avoid, how to close.

Example:
"Direct, no-fluff tone. Speak like a fellow B2B operator. Never start with 'Great question' or 'Thanks for reaching out'. Keep replies to 2–3 sentences. Always end with a soft CTA for a 15-min call."

After You Create the Client

  1. Click the client card to open its details
  2. Copy the SmartLead Webhook URL and paste it into your SmartLead campaign under Webhooks → reply-received trigger
  3. Copy the HeyReach Webhook URL and paste it into your HeyReach campaign webhook settings
  4. Send a test reply to verify the flow: webhook → classification → Slack approval message
